# AGENTS.md

## JavaScript

Zona para comportamiento de frontend compilado con Vite.

## Reglas

- Mantener el JS ligero; la app actual es mayoritariamente Blade + Filament.
- No introducir frameworks frontend grandes sin una decision explicita.
- Si se agrega interactividad, que degrade bien y no rompa formularios Blade.
- Mantener imports y puntos de entrada alineados con `vite.config.js`.

## Prioridades

- soporte responsive
- mejoras progresivas
- scripts pequenos y mantenibles
